# Ord (`.ord()`)

| type | returns |
| ---- | ------- |
| Method | `number` |

## Arguments
| name | type | description |
| ---- | ---- | ----------- |
| index | int, default: 0 | The index within the string in which to determine the character code of

## Explanation
Returns the character code of the character at the given index

## Examples

```swift
"a".ord() == 97
"abc".ord() == 97
"abc".ord(1) == 98
```
